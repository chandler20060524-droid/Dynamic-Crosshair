# Dynamic-Crosshair
A desktop python application that creates a customizable overlaying dynamic crosshair, whose color changes based on its environment to optimize attention to the crosshair in competitive shooter games.

To use:
  1. Download the entire "Dynmaic Crosshair V2" directory somewhere on your PC (you may encounter a warning which claims that this application is a virus,      well, it isn't. Still, run the program at your own volition.)
  2. Create a desktop shortcut of the .exe file or pin it at start for convenience.
  3. Run the program. You will see a command prompt. Just ignore it. The crosshair will always stay on top.
  4. Press F5 to exit, F6 to change size, F7 to change thickness.
